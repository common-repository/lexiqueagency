== Description ==

Plugin that adds a new type post named 'lexique_agency', a LEXIQUE shortcode in a page is added to the implement the search words by letters or by word.
The administration will be able to choose the type of search (normal or ajax).

This plugin requires jquery, we use the default library jquery includes in wordpress

1. New custom post type 'lexique_agency'.
2. ShortCode 'LEXIQUE' to add a page.
3. Ability to set the number of letters in search of a word.
4. Change colors of the button to search and the background of the list of letters.
5. Change colors of the current lette

== Installation ==

1. Download the plugin.
2. Upload the lexiqueAgency folder to your site's wp-content/plugins directory.
3. Active the plugin in the Wordpress Panel.
4. Start creating new lexique.
5. Setting's Page (Setting -> Glossary Option )
6. Create a page, then put the shortcode [LEXIQUE]
7. Visit the page created
